# 🐱💜 NeuroEva

**Локальный Telegram-бот — девушка-кошка с характером, памятью, эмоциями и генерацией изображений.**

> **Внимание!** Данный репозиторий не подлежит коммерческому использованию. Проект создан в личных целях и распространяется как есть.

---

## ✨ Особенности

- 🧠 **Общение через локальную LLM** — Qwen 2.5 7B (инструктивная модель)
- 🎨 **Генерация аниме-селфи** — NablaThetaA5 + апскейл через Upscayl
- 👁️ **Описание фото** — BLIP (анализ изображений)
- 📓 **Дневник и память** — SQLite + Markdown-дневник
- 🎭 **38 эмоций** — смена настроения, автономные сообщения, инициатива
- 👭 **Подруги** — Виолетта (готическая кошка) и Агнес Такион (девушка-лошадь, вдохновение из Umamusume)
- 📍 **Отслеживание локации** — влияет на фон генерации и доступность подруг
- 💬 **Автономность** — Ева может скучать, обижаться, предлагать показать подруг

### **🦾Установка происходит только на Arch-подобных дистрибутивах:**
 1. Скачайте папку репозитория в корневой раздел командой:
 `git clone https://github.com/MitiKagura/NeiroEva`
 2. Если хотите фотографии улучшенного качества, скачивате upscayl-bin через AUR. Например:
 `yay -S upscayl-bin`
 3. Перейдите в папку репозитория и откройте *install.sh*. Можно так же командой:
 `cd ~/NeiroEva && chmod +x install.sh && ./install.sh`
 - Установщик автоматически:
    
    - Установит системные зависимости

    - Создаст виртуальное окружение evavenv

    - Установит все Python-пакеты из requirements.txt

    - Скачает модель NablaThetaA5 (около 3 ГБ)
 
 4. Загрузите модель:
    - qwen2.5-7b-instruct-q4_k_m.gguf [👉Скачать](https://huggingface.co/paultimothymooney/Qwen2.5-7B-Instruct-Q4_K_M-GGUF/resolve/main/qwen2.5-7b-instruct-q4_k_m.gguf?download=true). Копируйте из загрузок в папку models
 5. Создайте файл .env
 `nano ~/NeiroEva/.env`
 - и впишите
`BOT_TOKEN=ваш_токен_от_BotFather
CREATOR_ID=ваш_telegram_id #Можно получить в боте @userinfobot`
 6. Запускайте командой:
`cd ~/NeiroEva && source ~/NeiroEva/evavenv/bin/activate && python.py`
    - Если у вас оболочка терминала fish:
`cd ~/NeiroEva && source ~/NeiroEva/evavenv/bin/activate.fish && python main.py`
